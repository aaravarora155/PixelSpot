Running the code

  1. CD into the directory where the file is stored by running the command: cd "PathToDirectory"
  2. Install the EJS dependency by using: npm i ejs
  3. Install any other dependencies listed on the top of each file
  4. Once you are in the directory run the command: node "appname.js"
