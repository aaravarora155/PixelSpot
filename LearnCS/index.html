How to join Google Classroom

Method #1:

Go to your email (the one you entered in the registration form), and you should have an email from aarav2607@gmail.com which will contain a link to join Google Classroom. If this method does not work for any reason, please try Method #2.

Method #2:

If the link does not work, navigate to Google Classroom and click the add icon in the top right corner. A dialog box should appear, asking you to enter the Class Code. The class code will be in the registration confirmation email. Then press Join. If you have any issues, navigate to our Support Page to contact us.
